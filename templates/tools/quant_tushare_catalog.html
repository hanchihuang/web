使用规则

只开放非分钟数据。分钟类 API 仍不在本次 relay 范围内。
API 客户端请求 JSON 目录时，继续访问 /tushare/pro/catalog 并携带 X-API-Key。
浏览器直接打开 /tushare/pro/catalog 时，会优先看到这个前端目录页。
